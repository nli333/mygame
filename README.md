# mygame
# 基于FTXUI 终端贪吃蛇小游戏
## 项目简介
本项目使用C++语言搭配FTXUI终端UI库开发控制台贪吃蛇游戏。
项目采用模块化编写，代码拆分出头文件、功能实现文件、主函数文件，结构规范清晰，方便后期代码维护与修改。

## 文件结构
1. snake.hpp ：项目头文件，定义游戏结构体以及全部自定义函数声明
2. snake.cpp ：游戏逻辑实现，包含蛇移动、食物生成、死亡判断功能
3. main.cpp ：程序入口，负责游戏界面绘制、键盘按键监听、自动移动计时器

## 游戏玩法
1. 使用上下左右方向键控制贪吃蛇移动
2. 吃掉红色食物可以增加分数并且变长身体
3. 触碰围墙或者撞到自身身体将会重新开局
4. 按下ESC按键可以直接退出程序

## 编译运行方式
1. 创建out编译文件夹，打开终端
2. 使用 cmake .. 加载配置文件
3. 编译生成可执行程序，直接运行即可游玩

## 贡献指南
1. 如需修改游戏速度、地图大小可以直接在头文件更改参数
2. 后续可以新增难度选择、游戏暂停等拓展功能
